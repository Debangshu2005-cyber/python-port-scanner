# Python Port Scanner

A simple Python Port Scanner built using Python in Kali Linux.

## Features
- Scan open ports on a target host
- Beginner friendly cybersecurity project
- Fast TCP connection scanning

## Installation

Clone the repository:

git clone https://github.com/Debangshu2005-cyber/python-port-scanner.git

## Usage

Run the scanner:

python3 port_scanner.py
